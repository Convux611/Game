خطوة واحدة وتنتهي — مجاناً تماماً
🔥 Firebase Realtime Database — مجاني 100%

هذه الإعدادة لازم تعملها مرة واحدة بس عشان اللعبة تشتغل بشكل موثوق مع أصدقائك

1
اذهب إلى console.firebase.google.com وسجّل دخول بحساب Google
2
انقر Add project (أو Create project)، اختر أي اسم، وأكمل الخطوات (انقر Continue لكل خطوة)
3
من القائمة الجانبية اختر Build → Realtime Database، ثم انقر Create Database
4
اختر أي region، ثم في الخطوة التالية اختر Start in test mode ثم Enable
5
انسخ رابط قاعدة البيانات — يبدو هكذا:
https://اسم-مشروعك-default-rtdb.firebaseio.com
6
افتح ملف index.html في أي محرر نصوص، وابحث عن هذا السطر في أول الكود:
const FIREBASE_URL = '';

وضع الرابط بين علامتي الاقتباس هكذا:
const FIREBASE_URL = 'https://مشروعك-default-rtdb.firebaseio.com';
7
احفظ الملف وارفعه على GitHub — اللعبة ستعمل بشكل مثالي الآن! 🎉
✅ بعد الإعداد لن تواجه مشكلة "انتظار الخوادم" مرة أخرى!
